面向新手的 Codex 配置教程

01

第一步：安装 Node.js

选择你的电脑系统，优先使用国内镜像安装 Node.js 与 npm，并在同一步完成验证。

  1. 打开 macOS 的“终端”。
  2. 复制右侧整段脚本并执行，按提示输入电脑密码。
  3. 脚本会自动检查 Homebrew，缺失时先从清华镜像安装，再继续安装 Node.js。
  4. 等待终端输出 Node.js is ready。

安装 Homebrew 与 Node.js 均优先使用清华镜像。期间如果终端要求确认，请按提示输入 Return 或电脑密码。 

export HOMEBREW_BREW_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git"
export HOMEBREW_CORE_GIT_REMOTE="https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/homebrew-core.git"
export HOMEBREW_INSTALL_FROM_API=1
export HOMEBREW_API_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles/api"
export HOMEBREW_BOTTLE_DOMAIN="https://mirrors.tuna.tsinghua.edu.cn/homebrew-bottles"
export HOMEBREW_PIP_INDEX_URL="https://pypi.tuna.tsinghua.edu.cn/simple"

if ! command -v brew >/dev/null 2>&1; then
  git clone --depth=1 https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/install.git /tmp/brew-install
  /bin/bash /tmp/brew-install/install.sh
  eval "$(/opt/homebrew/bin/brew shellenv)"
fi

brew update
brew install node
node -v
npm -v
node -e "console.log('Node.js is ready')"

  3. 安装结束后重新打开 PowerShell。
  4. 运行验证命令，确认 Node.js 可以执行脚本。

  2. 先把系统软件源切换到国内镜像，例如 清华 Ubuntu 镜像帮助 或对应发行版镜像。
  3. 更新软件源并安装 Node.js 与 npm。

02

第二步：安装 Codex App

访问 codex-app-mirror releases 下载最新版安装包，并按当前系统完成安装。

  1. 打开 Codex App GitHub Releases，选择标记为 Latest 的版本。
  2. 在 Assets 中下载与系统匹配的安装包。
  3. 运行安装包并按系统提示完成安装。
  4. 安装完成后打开 Codex App，确认应用能正常启动。

Windows 如果安装 .Msix 时提示“你的系统管理员已阻止此程序”，通常是当前设备策略不允许从商店外安装 MSIX / AppX 包，或应用安装器 / AppX 部署服务被禁用。请优先尝试 Microsoft Store 官方页面；公司、学校或其他受组织策略管理的设备需要联系设备管理员放行，GitHub Release 安装包不会绕过本机安装策略。

03

第三步：安装 Codex++

访问 codex++ releases 下载最新版安装包，并按当前系统完成安装。

  1. 打开 Codex++ GitHub Releases，选择标记为 Latest 的版本。
  2. 在 Assets 中下载与系统和芯片架构匹配的安装包。
  3. 运行安装包并按系统提示完成安装。
  4. 安装完成后重新打开 Codex 或终端，再检查 Codex++ 是否正常加载。

macOS 如果提示无法打开来自互联网的应用，请在“系统设置 → 隐私与安全性”中允许打开对应应用。

04

第四步：注册账号

前往 嘉合杉升AI接入系统 完成注册。注册时用户名请使用真名，方便管理员识别并给予额度。

  1. 打开注册页面并按提示填写账号信息。
  2. 用户名填写本人真名，避免使用昵称或缩写。
  3. 提交注册后等待管理员审核并分配额度。
